tinyproxy Release Order
-----------------------

Below is a list of all the tinyproxy releases since 1.3.2.  Each version
has its release date in brackets.  To find the information in the
ChangeLog do a:

    grep 'Released tinyproxy x.y.z'

and you will be brought to the entry for the release.

NOTE: Right now the list is not complete (or necessarily accuracte)
      between version 1.4.1 and 1.4.3.  I will be updating this
      list as more information becomes available.

      Also, there is currently no information for releases before
      version 1.3.2
